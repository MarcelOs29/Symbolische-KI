# Symbolische-KI
Dieses Projekt verknüpft ein logisches Schlussfolgerungssystem (symbolische KI) mit einem großen Sprachmodell (LLM, subsymbolische KI). Ziel ist es, natürliche Sprachprobleme in eine formale Repräsentation zu überführen, diese mit einem logischen Solver zu lösen und die Lösung wieder in natürlicher Sprache auszugeben. Die LLMs werden dabei lokal ausgeführt.

Detallierte Dokumentation findest du in der [doc/Documentation.md](doc/Documentation.md).

Die Tests die während der Entwicklung genutzt wurden, befinden sich direkt in der [src/main.py](src/main.py).
